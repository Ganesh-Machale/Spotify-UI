🎵 Spotify Web Player Clone

Hi 👋
This is my Spotify Web Player Clone built using HTML and CSS.

I created this project to practice frontend layout design, UI structuring, and responsive styling. The goal was to recreate the look and feel of Spotify’s web player interface as closely as possible.

🚀 About the Project

This is a static frontend clone of the Spotify web player.
It focuses mainly on layout, styling, and UI design.

The structure is built in:
index.html 

index

The styling is handled in:
Style.css 

Style

🛠 Technologies Used

HTML5

CSS3

Flexbox

Google Fonts (Montserrat)

SVG Icons

✨ Features

✔ Dark themed Spotify-style UI
✔ Sidebar with Home, Search, and Library
✔ Recently Played section
✔ Trending & Featured cards
✔ Sticky top navigation
✔ Fixed bottom music player layout
✔ Hover effects for cards and buttons
✔ Responsive layout adjustments

💡 What I Learned

While building this project, I improved my understanding of:

Creating complex layouts using Flexbox

Designing reusable card components

Building a sidebar + main content structure

Using position: fixed and sticky

Improving UI using hover effects and transitions

This project strengthened my frontend fundamentals.
